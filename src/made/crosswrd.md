<h1 class="log-post-title">crosswrd.site</h1>
<p class="meta">feb 2026</p>

<div class="prose">

I was browsing open source projects on GitHub and stumbled on [wordamour.com](https://wordamour.com/) — a word puzzle you make for someone you love. I thought: what if the same idea, but as a crossword?

That became crosswrd. You write the clues, it generates the crossword, you send them the link. Used the Excalidraw font for the UI because I love how it feels — hand-drawn, imperfect, warm.

Shipped it just in time for Valentine's Day. ♥

The name has a story too. I wanted crossword.us, but .us domains need American address verification. Then crossword.site — too expensive. So I dropped a letter and landed on crosswrd.site. Sometimes constraints name things better than you could.
